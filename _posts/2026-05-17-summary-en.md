The δ-mem paper proposes an efficient online memory mechanism for large language models, which augments a frozen full-attention backbone with a compact online state of associative memory. This is significant because it addresses the limitations of fixed-size state models with massive token history, enabling efficient online memory for large language models. The δ-mem mechanism provides low-rank corrections to attention computations, enabling efficient online memory for large language models.
